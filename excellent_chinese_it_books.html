推荐精品国产技术书籍

yalewoo 最后修改于 2015-07-18 发表于 2015-07-18 833 0

电子书
[目录开关]
  • 1汇编语言（第2版） 王爽 清华大学出版社
  • 2计算机网络 （第6版） 谢希仁 电子工业出版社
  • 3Linux C编程一站式学习

虽然目前国产技术书籍整体的名声不是很好，但其中还是有非常优秀的作品的。雅乐网给大家整理了一些值得推荐的国内书籍。

汇编语言（第2版） 王爽 清华大学出版社

这本汇编语言的书籍广受好评，豆瓣评分9.3分，是国产技术书籍中难得的高分。它的作者王爽老师既是一位计算机科学教育家，又是一位哲学家。书中的内容娓娓道来，循序渐进，每次讲解一个知识点，不会出现知识的跳跃。书中还有大量详细的截图，配上详细的讲解，让汇编学习不再困难。

目前有了第三版，京东链接：《汇编语言（第3版）》(王爽)【摘要 书评 试读】- 京东图书

计算机网络 （第6版） 谢希仁 电子工业出版社

国产难得的教材。计算机网络的知识有点繁杂，但本书的介绍非常清晰，从全局的网络宏观架构到一个个协议的细节，都娓娓道来，让人流连忘返。本书的一大特点是术语用词特别准确严谨，看下截图就知道了0.0

京东链接: 《“十二五”普通高等教育本科国家级规划教材：计算机网络（第6版）（附CD光盘1张）》(谢希仁)【摘要 书评 试读】- 京东图书

还有配套的习题解析  由于第六版和第五版的习题没变化，可以用第五版的习题解析

《计算机网络释疑与习题解答（第5版配套使用）》(谢希仁)【摘要 书评 试读】- 京东图书

Linux C编程一站式学习

这本书把系统结构和c语言的知识融合在了一起，知识面很广，包含了c语言基础，c语言进阶和linux编程的相关知识，适合有一定基础的人。

这本书有在线版本。

Linux C编程一站式学习： http://akaedu.github.io/book/

作者还正在写一个新版本：中文版：Linux C/C++编程一站式学习 — akabook 3.0 documentation     http://songjinshan.com/akabook/zh/index.html
