  • vc6添加文件到工程的时候错误

微软针对这个问题发布了一个补丁（http://support.microsoft.com/kb/241396），下载下来自解压后，在vc6里编译一下即可使用。然后按以下步骤:

1. 把编译后生成的FileTool.dll文件拷贝至 ..\Microsoft Visual Studio\Common\MSDev98\AddIns 文件夹下

2. 在cmd下手动注册该dll文件。使用注册命令”regsvr32.exe” + add-in.dll，比如雅乐网测试时的完整命令为：

regsvr32 “C:\Program Files (x86)\Microsoft Visual Studio\Common\MSDev98\AddIns\FileTool.dll”

3. 打开VC6，右击工具栏，自定义… -> 附加项和宏文件，选中 FileTool Developer Studio Add-in 附加项
这个时候就会出现 A 和 O 的文件栏按钮

Add File To Project

Open

4. 打开VC6->工程->定制->编辑器选择“主要”，移去FileOpen的快捷键Ctrl+O，接下来，类别->Add-ins，给这个OpenFile设置快捷键Ctrl+O；同样的方法，可以给 “添加文件”设置快捷键Ctrl+A。

5.VC6->工程->定制->工具栏，然后打开工具栏上面的“文件”，用鼠标拖着移去“打开”按钮的图标，接着把O用鼠标拖到“打开”的位置；同样的方法设置A到“工程”下的相应位置。

  •  安装VC++6.0行号显示工具

下载一个VC6LineNumberAddin.dll的组件,下载地址：http://pan.baidu.com/s/1nt7ENnr

注意:请不要按照一些行号插件里面的说明,那些说明只适用于WinXp系统的安装.

1. 如果你的VC安装在C盘，请拷贝文件VC6LineNumberAddin.dll到如下目录：

C:\Program Files(x86)\Microsoft Visual Studio\Common\MSDev98\AddIns

2.在cmd下手动注册该dll文件。参照上文

3. 注册

然后将下面文件内容复制到TXT文档中并改名为VC6LineNumberAddin_x64.reg,记住这个后缀是*.reg,这个文件是写入注册表的:

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\DavidHowe Software\Lienumber\]

“RegSN”=”aj3423”

3. 启用

确保你有administartor的管理员权限然后打开vc6，菜单栏：Tools -> customize -> Add-ins and Macro Files

选中VC6LineNumber Developer Studio Add-in

注意:如果没有管理员权限你的组件就会注册失败,这个行号组件就用不了.
