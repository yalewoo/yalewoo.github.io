MOOC平台可以让人们可以免费自主地学习心仪的大学课程。本文2016年4月更新，介绍一些我觉得质量比较高的计算机课程。如果大家发现了其他好课，欢迎在评论中推荐，让更多人看到。

概述

重写本文时，我尝试结合自身经验和网上观点，按照一个理想中计算机本科生应该掌握的技能为线索分类各种课程。可能有部分课程不是MOOC形式的视频课程。

 

下面表格中的授课老师由于版幅的原因，只写了课程介绍中第一个老师。课程前面的是序号，并不是排名。

一、数学基础

1. 微积分

1.1 微积分B(1) 微积分B(2) ，清华大学 ，扈志明，章纪民 自主模式

来自清华的微积分（B）主要面向清华大学理工科专业的学生，部分内容比一般工科数学要深一点。老师水平超高，讲课思路清晰，引入，定义，定理，证明，例题，一气呵成，几乎完美。但由于难度的原因，如果数学基础不好可能要听很多遍才能学会，我个人考研的时候差不多看了3遍，每次都感觉收获很大。作业习题有一定的难度。

这门课还有自助学习模式，可以自由安排进度，但目前自助模式不能提供证书。

另外，清华大学面向高中生推出了 微积分（先修课），这个我个人没有看过，如果你对上面的课程感觉困难很大可以试试这个，先入门了解大概后再深入学习也是极好的。

1.2 微积分启蒙， 北航，李尚志 2016.2.29-2016.6.20

介于通识课程和专业课程之间，用新的视角看待微积分。这门课致力于把你领进门，感受微积分的思维，引起你对微积分的兴趣，令人获益匪浅。

另外，李尚志老师的数学大观课程广受好评，有让人爱上数学的神奇功效，不可错过。

1.3 考研数学视频

如果你的基础和我一样实在太差，上面的两门课都感觉困难，可以试试考研视频。因为考研的视频面向的是没有任何基础的各种学生，听起来也会很轻松。

推荐汤家凤老师或者张宇老师的基础班视频，可以自行百度，入门后可以再看上面的两门深入学习。

2. 线性代数

2.1 线性代数，山东大学，秦静，  2016.3.15-2016.6.15

中规中矩的线性代数课，授课内容和工科考研考纲一致，包含行列式，矩阵，向量，线性方程组，相似对角型和二次型六部分内容。授课老师思路清晰易懂，即可以帮你轻松掌握考研内容，又可以为后面深入学习打下坚实的基础。

另外还有配套的线性代数解题技巧及典型题解析课程，简直是考研学生的福利有木有。

2.2 线性代数启蒙，北航，李尚志，2016.2.29-5.13

从一个新的角度理解线性代数，引导你入门。

2.3 线性代数(1)  线性代数(2) ，清华大学，马辉， 自主安排学习进度

似乎是面向理科，有许多工科没有涉及到的知识点。可以用来提高。

另外清华还有面向高中生的线性代数（先修课）（2016春）-学堂在线慕课(MOOC)平台

2.4 麻省理工公开课：线性代数 Gilbert Strang，视频

传说中一门让你真正理解线性代数的神课。

2.5 线性代数与空间解析几何（一） （二），电子科技大学，黄廷祝，2016.2.22-7.3

刚发现的一门课程，将代数与几何结合，代数为几何问题的解决提供方法，代数又在几何空间的基础上得到延伸与扩展。看介绍应该不错。

二、编程语言

1. C语言

1.1 浙大，翁恺， 程序设计入门——C语言 ，C语言程序设计进阶 ，已结束可查看

广受好评的c语言入门课，老师非常用心，讲解全面细致严谨，让人学到很多。

1.2 北大， 李戈 ，计算导论与C语言基础， C程序设计进阶， 自主学习

同样经典的c入门课。

1.3 清华，张悠慧，汇编语言程序设计，自主学习

讲解汇编语言和c，对理解c语言和计算机底层帮助非常大。参考书籍《深入理解计算机系统》。

2. C++

2.1 北大，刘家瑛、郭炜   C++程序设计，自主学习

刘家瑛和郭炜老师的课，原名叫程序设计实习。课程相当风趣欢乐，同时也能帮助大家快速入门c++。话说郭老师扔瓶子的时候很霸气呀！

2.2 浙大，翁恺，面向对象程序设计-C++

据说浙大内选不到翁老师课的同学都会到网上看。。。

2.3 清华，郑莉，C++语言程序设计基础，C++语言程序设计进阶，自主学习

清华大学的c++基础和进阶，中规中矩，讲解完整清晰，虽然有时略显枯燥，但坚持学习下来更帮助我们系统地掌握c++。

2.4 北邮，崔毅东，C++程序设计入门(上)，C++程序设计入门(下)，已结束可查看

北邮的C++程序设计也非常不错，很有特色，值得一看。开课的时候很多活动，可惜现在已经结束，不过视频仍然可以看。还有QQ群老师答疑。

2.5 清华，徐明星，面向对象程序设计（C++），2016春

课程使用的语言是C++，在通过一定篇幅介绍C++的主要语法知识后，将重点围绕什么是接口、怎么设计、实现和使用接口、如何在程序设计中体现“开闭”原则，以及如何实现数据表示与操作的分离等内容展开介绍和讨论。

三、数据结构和算法

1. 数据结构

1.1 清华，邓俊辉，数据结构，自主学习、2016春（上） （下）

超级推荐！不愧是清华的数据结构课，邓老师的数据结构讲解思路非常清晰，让人能感觉到算法和数学的魅力。配套ppt和板书效果超赞。不仅讲解实现思路，还有具体的c++代码实现。内容相比一般院校的数据结构要更广更深，对于B树、红黑树、串匹配等算法有很详细的介绍。编程练习有一定的难度。

建议看一下课程介绍视频，可以感受到这门课的魅力。

1.2 浙大，陈越，数据结构，已结束可查看

浙大陈姥姥的数据结构课程也非常不错，内容接近考纲要求，具体代码使用c语言。

1.3 北大，张铭，数据结构基础，高级数据结构与算法，自主学习

北大的数据结构课程是Coursera推出的程序设计与算法 专项课程中的部分课程，也很值得推荐。

2. 算法

2.1 北大， 刘家瑛、郭炜， 算法基础 ，自主学习

刘家瑛和郭炜老师的算法基础课，和其他算法课程不一样。这门课课堂上就是讲解的算法题目，如何思考以及怎么实现，听完这门课你做编程题目（OJ）的能力会急剧上升。包含探索枚举、二分、贪心、递归、深度优先搜索、广度优先搜索、动态规划等经典算法。

2.2 北大， 屈婉玲， 算法设计与分析 ，已结束可查看

曲奶奶的算法设计与分析课程感觉偏向理论一点，讲解的内容也很多。

2.3 哈工大 王宏志  算法系列课程

哈工大的算法系列课程是一个系列的，每章内容差不多三个视频，虽然很少但核心思想都讲到了。

2.4 计算几何 邓俊辉 自主学习、2016春

邓俊辉老师的新课，主要讲解计算机处理几何问题的算法，值得一听。应该不少人奔着邓老师选这门课的←＿←
