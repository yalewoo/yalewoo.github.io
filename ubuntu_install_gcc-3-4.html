ubuntu安装低版本gcc-3.4（gcc-3.4-ubuntu.tar.gz百度云下载地址）

[目录开关]
  • 1安装gcc-3.4

由于编译linux0.11内核需要使用这个版本的gcc，所以需要安装。

安装gcc-3.4

由于apt-get提示找不到这个版本，所以下载deb安装包进行安装。

需要这些文件

cpp-3.4_3.4.6-6ubuntu3_i386.deb

g++-3.4_3.4.6-6ubuntu3_i386.deb

gcc-3.4-base_3.4.6-6ubuntu3_i386.deb

gcc-3.4_3.4.6-6ubuntu3_i386.deb

libstdc++6-dev_3.4.6-6ubuntu3_i386.deb

下载地址：

链接: http://pan.baidu.com/s/1eQgZkw6 密码: cbb8

解压后得到文件夹gcc-3.4，里面有上面这些deb文件。cd进去之后输入

1
sudo dpkg -i *.deb

就可以进行安装了。
