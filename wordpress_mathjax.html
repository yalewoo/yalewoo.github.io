wordpress使用mathjax生成数学公式

yalewoo 最后修改于 2015-05-01 发表于 2015-05-01 1,826 0

wordpress wordpress
[目录开关]
  • 1使用方法

MathJax 是一个开源的基于 Ajax 的数学公式显示的解决方案，结合多种先进的Web技术，支持主流的浏览器。MathJax 根据页面中定义的 LaTex 数据，生成对应的数学公式。

官网：MathJax

http://www.mathjax.org/

本文介绍WordPress下如何使用。具体用法可以参考

官方文档

Mathjax与LaTex公式简介 // ML works

一份不太简短的latex介绍

使用方法

可以直接使用mathjax的服务器，也可以本地搭建。

直接在WordPress文章编辑时切换到文本状态，在最上面添加一行

1
<script type="text/javascript" src="https://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-AMS-MML_HTMLorMML"></script>

就可以了。注意不能有换行，否则WordPress会自动添加段落标签导致导入css失效。

显示效果如下

$$\frac{1}{3}$$
