七牛云外链图片地址改变

yalewoo 最后修改于 2015-08-04 发表于 2015-08-04 1,061 0

wordpress
[目录开关]
  • 1修改已有的图片

之前雅乐网介绍过使用七牛云作为wordpress博客的图床。

使用的域名前缀是

用户名.qiniudn.com

现在据说qiniudn.com的域名有点问题，刚才访问的时候显示域名不存在。所以推荐大家使用七牛推荐的域名。

在七牛空间设置 域名设置中 可以看到你的七牛域名 比如雅乐网的是这样的

 

需要在http://你的博客域名/wp-admin/options.php中修改

upload_url_path部分

修改已有的图片

登陆你的主机 空间管理中的phpAdmin，执行替换语句就可以了

1
2
3
UPDATE wp_posts SET post_content = REPLACE( post_content, '原域名', '新域名');
#示例：
#UPDATE wp_posts SET post_content = REPLACE( post_content, 'http://yalewoo.qiniudn.com', 'http://7d9rd6.com1.z0.glb.clouddn.com');
