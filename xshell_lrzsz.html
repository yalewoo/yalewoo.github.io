xshell使用lrzsz在远程linux和本地windows直接传输文件

yalewoo 最后修改于 2016-03-28 发表于 2014-10-29 3,830 0

Linux linux

如果是centos系统，可以用yum -y install lrzsz命令安装。

装好后

可以使用rz，sz命令进行ZModem文件传输

windows端需要支持ZModem的telnet/ssh客户端(xshell支持)

运行命令rz，即是接收文件，xshell就会弹出文件选择对话框，选好文件之后关闭对话框，文件就会上传到linux里的当前目录

运行命令sz file 就是发文件到windows上。比ftp命令方便多了，而且服务器不用再开FTP服务了。
