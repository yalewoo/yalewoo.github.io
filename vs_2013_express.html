Microsoft Visual Studio Express 是微软集成开发环境软件 Visual Studio 产品的轻量版本，而且是完全免费的。

如果你承受不起VS的高昂价格，又不想使用盗版，那么这个express版本可以试试哦。虽然比专业版少了一些功能，不过对于初学者来说是完全足够的。

授权形式

有同学问Express版本能不能用于商业开发，雅乐网到官网上看了一下，是这样说的

英文版

Commercial use of Visual Studio Express

Visual Studio Express products are available at no charge and may be used for commercial, production usage subject to the license terms provided with each product. For example, you can use Express for Windows to create apps that you can then submit for sale in the Windows Store.

中文版

Visual Studio Express 的商业用途

Visual Studio Express 产品是免费提供的，可用于商业和生产用途，但受每种产品随附的许可条款的约束。例如，您可以使用 Express for Windows 创建可在 Windows 应用商店中提交出售的应用程序。

看以看出，微软是允许使用express版本进行商业用途的，不过受到许可协议的限制，雅乐网找来了Express 2013的许可协议，这是雅乐网安装的时候的许可协议，我把它上传了自己的服务器上

http://www.yalewoo.com/attach/Microsoft_License_Terms.htm

大家自己看吧。。。反正雅乐网没看到什么限制QAQ
